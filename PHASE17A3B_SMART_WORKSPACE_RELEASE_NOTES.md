# Phase 17A.3B – Smart Workspace

Built on Phase 17A.3A.

## Added
- Unsaved changes detection for forms.
- Warning before closing or leaving a dirty form.
- Pinned tabs.
- Reopen Last Closed tab.
- Improved Windows menu with reopen action.
- Keyboard shortcuts:
  - Ctrl+W: close current ERP tab
  - Ctrl+Shift+T: reopen last closed ERP tab
  - Ctrl+Tab: switch to next ERP tab
  - Ctrl+K: focus search
- Dirty tab indicator using `*`.
- Windows counter continues to show open forms.

## Notes
- This phase changes UI/navigation only.
- No database structure changes.
- No authentication changes.
- No business logic changes.
